## Features
- Built with **React.js** for dynamic UI.
- Smooth **animations** using Framer Motion & CSS.
- Fully **responsive** design (desktop, tablet, mobile).
- **Dark/Light mode** toggle.
- Sections:
  - **Home:** Animated header & introduction
  - **About Me:** Skills, experience, education
  - **Portfolio:** Projects with hover effects & animations
  - **Contact:** Email, WhatsApp, social links with interactive buttons

---

## Tech Stack
- **Frontend:** React.js, JavaScript, HTML5, CSS3
- **Animations:** Framer Motion / React Spring / CSS transitions
- **Version Control:** Git & GitHub

---


## How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/kl0l89/portfolio.git
````

2. Navigate into the project folder:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm start
```

5. Open [localhost:3000](http://localhost:3000) in your browser.
